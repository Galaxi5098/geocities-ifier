# GeoCities-ifier 🌐

A Chrome extension that transforms any modern webpage into a 1999-era GeoCities masterpiece.

Applies retro visual effects including animated GIFs, marquee text, tiled backgrounds, visitor counters, comic sans, and all the classic web aesthetics your eyes have been missing.

## Install

Available on the [Chrome Web Store](#) *(link coming soon)*

**Manual install:**
1. Download or clone this repo
2. Go to `https://chromewebstore.google.com/detail/gbaojdjnklahlpjniaflcbefklhoenla?utm_source=item-share-cb`
3. Enable "Developer mode"
4. Click "Load unpacked" and select this folder

## How It Works

Click the extension icon on any page. Watch the future dissolve into the past.

## Permissions

- **activeTab** — Access the current page when you click the icon
- **scripting** — Inject the retro CSS and JS transformations

No data is collected, stored, or transmitted. See the [privacy policy](https://galaxi5098.github.io/geocities-ifier/privacy.html).
